I am a **Computer Science Engineering student** at Óbuda University, specializing in **Artificial Intelligence and Robotics**. My core focus lies in bridging the gap between high-level deep learning models and low-level system programming for autonomous systems. 

- 🔭 I’m currently working on: **Deep Learning Architectures & ROS 2 Integrations**
- 🌱 I’m currently diving deep into: **Robot Programming, Modern C++, and Sensor Fusion**

### 🚀 Featured Project
**[KUKA YouBot ROS 2 Bridge](https://github.com/MarkBenWagner/KUKA_YouBot_ROS2_Bridge)**
A cross-platform (WSL + Windows) distributed system establishing a low-latency UDP bridge between a modern ROS 2 network and legacy KUKA hardware. *Currently exploring this architecture and potential AI extensions as a candidate for my upcoming BSc thesis.*
